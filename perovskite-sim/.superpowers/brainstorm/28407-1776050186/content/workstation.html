What's in the default layout

  • Top toolbar — New Device wizard (tier picker lives here), Run, Run All, Compare, Sweep, Help
  • Project tree (left) — Devices with tier badges, experiments, runs, cross-device Results folder
  • Device Schematic — SCAPS-1D-style drawn stack, clickable layers feed the inspector
  • Main Plot — J–V / Nyquist / degradation depending on active experiment; older runs overlaid for time-comparison
  • Parameter Inspector — context-sensitive, gated by device tier
  • Secondary View — band diagram or metric cards, user can flip between them
  • Solver Console — always-visible bottom strip; first token is the active physics tier
  • Help — Tutorial / Parameters / Algorithm become content in a Help pane, dockable like everything else (hidden by default but one click away)

Things I'm deliberately leaving out of v1

  • No floating windows — all panes dock inside the main area (simpler, matches VS Code)
  • No device cross-section viewer (real 2D SEM-style rendering) — just the 1D stack schematic
  • No scripting console — the solver console is read-only log, not a REPL
  • No plugin system / custom-experiment loading
  • No multi-user / sync — workspace is a local file
  • No undo stack beyond "Reset" per device (can add later if needed)
