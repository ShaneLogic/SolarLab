A — Workstation

Multi-pane IDE layout. Project tree on the left, multiple result views tiled on the right, persistent console / solver log. Device is a first-class object you configure once, then run many experiments against.

Feels like: TCAD, Silvaco ATLAS, Sentaurus, Lumerical CHARGE

B — Structured Notebook

Keep the tab metaphor but make it feel like a real lab notebook. Big visual device schematic with clickable layers, experiments as tabs, each experiment shows its parameters alongside its results. Device state is shared across tabs.

Feels like: SCAPS-1D, modern PC1D, SolarCellSimu, AFORS-HET

C — Study Studio

Linear wizard-style workflow: Setup → Physics → Mesh → Solve → Results, with run history and parameter sweeps as first-class objects. No tabs — one active step at a time, but nothing gets lost.

Feels like: COMSOL Model Builder, Driftfusion Web, Ansys Granta
