# Phase F — PVK doping direction (Phase E1.15, 2026-05-27)

**Status:** investigation complete, no code change
**Branch:** `e1.15-pvk-doping-direction` (closing without commit)

## Goal

Close the PVK donor doping V<sub>oc</sub> DIRECTION mismatch on the SCAPS partner
sweep:

- SCAPS PDF: V<sub>oc</sub> rises with N_D_PVK (1.168 → 1.202 V across 1e8–1e18
  cm⁻³)
- SolarLab E1.6: V<sub>oc</sub> range matches (34 mV) but direction REVERSES —
  V<sub>oc</sub> falls with N_D_PVK (1.060 → 1.028 V across 1e12–1e18)

Hypothesis: activating HTL/PVK interface defect (mirror of the PVK/ETL
defect from Phase E1.5) flips direction by introducing band-bending
sensitivity to PVK doping at the HTL contact.

## Method

Three probes via `dataclasses.replace` on scaps_mirror.yaml:

1. **Baseline** (HTL/PVK off): measure current PVK doping direction
2. **HTL/PVK cf=1e-4** (matching PVK/ETL): test if direction flips
3. **HTL/PVK cf sweep** (1e-6, 1e-8, 1e-10): find weakest activation
   that flips direction while preserving CBO + baseline parities

## Results

### Probe 1 — baseline PVK doping sweep (HTL/PVK OFF)

| N_D_PVK [cm⁻³] | V<sub>oc</sub> | direction |
|---|---|---|
| 1e12 | 1.0605 | (baseline) |
| 1e14 | 1.0605 | flat |
| 1e16 | 1.0620 | +1.5 mV |
| 1e18 | 1.0280 | **−34 mV** |

Range 34 mV ✓ matches SCAPS, direction MISMATCH (high-doping V<sub>oc</sub>
collapse instead of rise).

### Probe 2 — HTL/PVK cf=1e-4 (mirror PVK/ETL)

| N_D_PVK | V<sub>oc</sub> | direction |
|---|---|---|
| 1e12 | 1.0605 | (baseline) |
| 1e14 | 1.0714 | +11 mV |
| 1e16 | 1.1165 | +56 mV |
| 1e18 | 1.1229 | +62 mV ✓ |

Direction MATCHES SCAPS. **BUT**:
- Baseline V<sub>oc</sub> = 1.0812 (was 1.069, OK)
- J<sub>sc</sub> = 333.53 A/m² (was 240 — **non-physical 39 % rise**; SCAPS max
  is 26.28 mA/cm² = 263 A/m²)
- PCE = 30.4 % (was 22.0 — non-physical, exceeds SQ limit for Eg=1.53)
- CBO sweep range = **261 mV** (was 782 mV — **66 % regression**)
- Cliff drop = 261 mV (was 290, modest regression)

HTL/PVK SRH at cf=1e-4 produces a non-physical J<sub>sc</sub> bump AND breaks the
CBO closure (Phase E1.5's primary win). Cannot ship.

### Probe 3 — weaker cf sweep

| HTL/PVK cf | baseline V<sub>oc</sub> | baseline J<sub>sc</sub> | CBO cliff drop | PVK doping direction |
|---|---|---|---|---|
| 1e-6 | 1.0696 ✓ | 240.55 ✓ | 281 mV ✓ | **mismatch** ✗ |
| 1e-8 | 1.0694 ✓ | 239.58 ✓ | 281 mV ✓ | **mismatch** ✗ |
| 1e-10 | 1.0694 ✓ | 239.57 ✓ | 281 mV ✓ | **mismatch** ✗ |

At ANY HTL/PVK cf weak enough to preserve CBO closure + baseline
physics, the PVK doping direction REVERTS to mismatch. The
direction-flip at cf=1e-4 was a SIDE EFFECT of strong HTL/PVK
recombination dominating bulk SRH — not a physics-driven fix.

## Diagnosis

The PVK doping direction mismatch is STRUCTURAL, same class as the
base V<sub>oc</sub> gap diagnosed in Phase G (E1.14):
- SCAPS may handle doping-dependent band bending differently (Φ<sub>b</sub>
  ohmic-equivalent contact BC vs SolarLab Dirichlet pin)
- SCAPS' degenerate-carrier statistics may matter at N_D_PVK > 1e16
  cm⁻³ where Boltzmann statistics depart from Fermi-Dirac

Same root cause as Phase G: closure needs SCAPS source-code access OR
implementation of Boltzmann-degenerate carrier statistics (Phase E3
research-grade if partner asks).

## Decision

**Defer PVK doping direction closure.** Document as known limitation in
partner report. Two reasons:

1. No simple parameter tune flips direction without breaking CBO closure
2. Real physics fix requires Phase E3 (Boltzmann-degenerate statistics)
   or Phase E2 (Φ<sub>b</sub> contact BC) — both multi-week refactors

The current 34 mV range matches SCAPS magnitude — partner sees the
correct sensitivity to PVK doping, just with the opposite sign. For
device-design use (e.g. "how does V<sub>oc</sub> vary if PVK doping doubles?")
this is partially useful but flagged as "direction TBD".

Move to **Step 8 (Phase H) — final partner report** with current parity
state.
