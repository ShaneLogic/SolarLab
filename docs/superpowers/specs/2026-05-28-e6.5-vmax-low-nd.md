# Phase E6.5 — Low-Nd ETL V_max extension gate

**Status:** FAILS the "V_max-only fix" hypothesis; opens E6.6 contact/branch audit
**Branch:** `e6.5-vmax-low-nd`
**Input:** `outputs/scaps_validation_e6_5_vmax/`

## What ran

E6.4 showed `Nd_ETL` had three low-doping points with `voc_bracketed=False`
at `V_max=1.6 V`, which made the old parked ETL-doping diagnosis unreliable.
E6.5 reran only the `Nd_ETL` sheet with a larger sweep ceiling:

```bash
cd perovskite-sim
python scripts/run_scaps_v2_regression.py \
  --sheets Nd_ETL \
  --v-max 2.5 \
  --out-dir ../outputs/scaps_validation_e6_5_vmax
```

The script now accepts `--sheets` and `--v-max` so this gate is reproducible
without one-off edits.

## Decision matrix outcome

The intended E6.5 matrix had three outcomes:

1. If the low-Nd points bracket and closure jumps to ~80 %, the new E6
   diagnosis is confirmed end-to-end.
2. If they bracket but closure stays ~30 %, V_bi shift is not the issue.
3. If they remain unbracketed at 2.5 V, suspect deeper contact-BC trouble.

The actual result is a fourth case: **most low-Nd points bracket, but on an
unphysical high-Voc branch**. This means the `V_max=1.6` sentinel artifact was
real, but raising `V_max` alone does not restore SCAPS parity. It reveals a
branch-selection/contact-equilibrium problem at very low ETL doping.

## Updated diagnosis

E6.4 correctly falsified the parked "immediate Newton-Krylov refactor" story:
the old 784 % over-sensitivity was contaminated by unbracketed V_oc sentinels
and wrong v1 defect inventory.

E6.5 adds a sharper constraint:

- The low-Nd region is not simply missing enough voltage range.
- SolarLab can converge to an excessive high-Voc branch when ETL doping is
  very low (`Nd_ETL=1e11-1e12 cm^-3`).
- The problem now points to contact equilibrium, built-in-potential handling,
  or branch selection under low ETL doping, not to the E2/E3/E4 interface-plane
  refactor path.

## Recommendation

Open **E6.6** as a narrow low-Nd/contact-BC audit:

1. Probe dark equilibrium and boundary carrier densities at `Nd_ETL=1e10-1e13`.
2. Compare `V_bi_eff`, contact densities, and terminal current sign near the
   bracketed root.
3. Test whether Robin/selective-contact settings or ETL contact pinning create
   the high-Voc branch.
4. Keep the E2/E3/E4 Newton-Krylov/QSS refactor parked unless E6.6 rules out
   contact-equilibrium and branch-selection explanations.
