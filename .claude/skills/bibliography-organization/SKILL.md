# Bibliography Organization and Sorting

Use this skill when organizing reference lists, bibliographies, or citation sections in academic papers.

## When to Use

- Creating or maintaining a bibliography section
- Organizing a reference list alphabetically
- Handling multiple publications by the same author
- Ensuring proper alphabetical sorting in academic documents

## Sorting Procedure

### 1. Primary Sort by Author Surname

Sort all entries alphabetically by the first author's surname. Group entries by the first letter:

- The chunk begins with 'S' surnames (Solomon, Sommerfeld, Spicer, Sze)
- The chunk transitions to 'T' surnames (Tabata, Tauc, Tersoff) after the last 'S' entry

### 2. Handle Multiple Works by Same Author

- Sort chronologically by year of publication
- Use letter suffixes (a, b, c) to distinguish multiple works by the same author in the same year

**Example:** W.E. Spicer entries appear in order: 1968, 1977, 1980a, 1980b, 1988

**Example:** D.L. Staebler (1977a) and D.L. Staebler (1977b)

### 3. Maintain Consistent Name Format

- Use consistent formatting throughout the bibliography
- Common formats: Initials followed by Surname (e.g., "W.E. Spear") or Surname followed by Initials
- Apply the same format to all entries

## Constraints

- Does not apply to numbering systems sorted by citation order
- Only applies when alphabetical organization is required
