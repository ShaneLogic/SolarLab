# Detailed Citation Format Examples

## Journal Article Format

### Standard Pattern
```
Author(s). Journal Name **Volume**, Page (Year)
```

### Examples
- J. Tersoff, D.R. Hamann, *Phys. Rev. B* **31**, 805 (1985)
- J.I. Pankove, D.E. Carlson, J.E. Berkeyheiser, R.O. Wance, *Phys. Rev. Lett.* **51**, 2224 (1985)
- J. Willer, H. Oppolzer, *Thin Solid Films* **147**, 117 (1987)

### Elements
- **Authors**: Initials followed by surname (J. Tersoff)
- **Multiple authors**: Separated by commas
- **Journal name**: Italicized, often abbreviated
- **Volume**: Bold notation (**31**, **53**)
- **Page**: Starting page number
- **Year**: In parentheses

## Book Format

### Standard Pattern
```
Author. *Book Title* (Publisher, City, Year)
```

### Examples
- E.D. Palik, *Handbook of Optical Constants of Solids* (Academic Press, New York, 1985)
- A.H. Wilson, *The Theory of Metals* (Cambridge University Press, Cambridge, 1954)
- K.T. Wilke, *Kristallzüchtung* (VEB, Berlin, 1973)

### Elements
- **Author**: Single author or editors
- **Title**: Italicized
- **Publisher**: Full publisher name
- **City**: Publication location
- **Year**: Publication year (may include letter suffix: 1984a, 1984b)

## Conference Proceedings Format

### Standard Pattern
```
Author(s), in Conference Name (Year), p. page
```

### Examples
- S.R. Ovshinsky, *Int. Conf. Struct. Excit. Amorph. Solids*, AIP Conf. Proc. **31**, 29 (1976a)
- B. Teyali et al., in *IEEE Photovolt. Spec. Conf.* (2002), p. 600

## Book Chapter Format

### Standard Pattern
```
Author(s), in *Book Title*, ed. by Editor Name (Publisher, City, Year), pp. range
```

### Examples
- T. Tiedje, in *Hydrogenated Amorphous Silicon II*, ed. by J.D. Joannopoulos (Springer, New York, 1984), pp. 261–300
- Y. Petroff, in *Handbook on Semiconductors*, ed. by T.S. Moss, M. Balkanski, *Optical Properties of Solids*, vol. 2 (North-Holland, Amsterdam, 1980)

## Special Notation

### Multiple Authors
- Three or fewer: List all authors
- More than three: Use "et al." after primary authors
- Example: M. Morgenstern et al. (2000a)

### Page Ranges
- Use en-dash (–) not hyphen
- Example: 291–301, 969–972

### Year Disambiguation
- Same author, same year: Add letter suffix
- Example: S.R. Ovshinsky (1976a), S.R. Ovshinsky (1976b)

### Volume Notation
- Always bold for journals
- Example: **31**, **53**, **QE22**
