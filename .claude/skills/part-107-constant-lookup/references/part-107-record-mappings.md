## Character Reference
Common special characters found in constant values:
- ε: epsilon
- ᣕ: special symbol
- −: minus symbol
- ⊥: perpendicular/up tack
- ×: multiplication symbol (used in record ID notation)

## Value Format Patterns
- Most values follow format: [number]=[alphanumeric code]
- Multiple values separated by semicolons
- Suffixes indicate units or classifications: 'm', 'c', 'sdnm', 'sdpm', 'plm', 'phm'
