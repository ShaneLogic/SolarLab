# Transition Equations Reference

## Intracenter Transitions

Intracenter transitions conserve charge at the impurity center:

```
M^n + hν → M^n*
```

Where:
- M^n is the impurity in charge state n
- M^n* is the excited state
- hν is the photon energy

Example (Eq 10.12 reference):
```
Cr³⁺ (³A₂) + hν → Cr³⁺ (³T₂)
```

## Ionization Transitions

Ionization transitions involve charge transfer:

### To Conduction Band
```
M^n + hν → M^(n+1) + e⁻
```

### From Valence Band
```
M^n + hν → M^(n-1) + h⁺
```

### To Another Center
```
M^n + hν → M^(n+1) + X^-
```

Example (Eq 10.13 reference):
```
Cr²⁺ + hν → Cr³⁺ + e⁻ (CB)
```

## Energy Considerations

- Ionization energy: E_ion = E(M^(n+1)) - E(M^n)
- Transition energy: E_trans = E(excited) - E(ground)
- Typical charge state differences: ~0.5 eV

## Selection Rules

### Spin Selection Rule
- ΔS = 0 (spin forbidden transitions are weak)

### Laporte Rule (Centrosymmetric Sites)
- g ↔ u transitions allowed
- g ↔ g or u ↔ u forbidden

### Exceptions
- Spin-orbit coupling relaxes spin rules
- Vibronic coupling relaxes Laporte rule
