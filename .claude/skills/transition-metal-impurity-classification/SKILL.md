# Transition Metal Impurity Classification

## When to Use
Use this skill when you need to characterize transition metal impurities in semiconductor hosts, including:
- 3d, 4d, 5d transition metals
- 4f rare earth elements
- 5f actinides
- Host materials: Si, III-V, II-VI semiconductors

## Classification Workflow

### 1. Determine Site Character

**Interstitial sites:**
- Typical for 3d elements in Si
- Zn to Ti group elements have minimal d-orbital interaction with host

**Substitutional sites:**
- Typical for 5d elements
- Involves interaction with vacancy states (hybridization)

### 2. Determine Charge Character

**By Oxidation State:**
- Based on valence electrons (e.g., Cr³⁺)

**By Charge Relative to Host:**
- Compare impurity charge to replaced host atom
- Example: Cr³⁺ replacing Ga³⁺ is neutral; Cr²⁺ is positive

**Multiple Stable Charge States:**
- Common when energy differences are small (~0.5 eV)
- Consider all thermodynamically accessible states

### 3. Determine Spin Character

**High-Spin Configuration:**
- Electrons preferentially parallel (Hund's rule)
- Indicates weak crystal-field interaction

**Low-Spin Configuration:**
- Electrons antiparallel
- Indicates strong crystal-field interaction

### 4. Determine Transition Character

**Intracenter Transitions:**
- Charge conserved at the impurity center
- See references for equation examples

**Ionization Transitions:**
- Charge transfer to band or other center
- See references for equation examples

## Output
Provide a complete impurity profile including:
- Site character (interstitial/substitutional)
- Charge state(s) and stability
- Spin configuration (high/low spin)
- Transition types possible
- Orbital configuration (e.g., 3d⁵ 4s¹)
- Symmetry (e.g., tetrahedral, orthorhombic)

## Key Considerations
- Crystal field strength determines spin state
- Energy differences between charge states are typically small
- Hybridization effects are significant for substitutional sites
