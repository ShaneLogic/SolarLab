# Lattice Parameters and Mismatch Calculations

## CdS Crystal Structure

CdS films deposited by CBD exhibit:
- Mixed cubic/hexagonal or predominantly hexagonal structure
- Grain size: order of tens of nanometers

## Lattice Parameters

### Cu(InGa)Se2 (112) Plane Spacings

| Composition | d(112) (nm) |
|-------------|-------------|
| CuInSe2 | 0.334 |
| CuIn0.7Ga0.3Se2 | 0.331 |
| CuIn0.5Ga0.5Se2 | 0.328 |

### CdS Plane Spacings

| Plane | Structure | d-spacing (nm) |
|-------|-----------|----------------|
| (111) | Cubic | 0.336 |
| (002) | Hexagonal | 0.336 |

## Lattice Mismatch Calculation

For pure CuInSe2:
```
Mismatch = (d_CdS - d_CIGS) / d_CIGS
Mismatch = (0.336 - 0.334) / 0.334 = 0.6%
```

For CuIn0.7Ga0.3Se2:
```
Mismatch = (0.336 - 0.331) / 0.331 = 1.5%
```

For CuIn0.5Ga0.5Se2:
```
Mismatch = (0.336 - 0.328) / 0.328 = 2.4%
```

## CdS Film Composition

CBD-deposited CdS typically:
- Deviates from stoichiometry (cadmium-rich)
- Contains impurities: O, H, C, N
