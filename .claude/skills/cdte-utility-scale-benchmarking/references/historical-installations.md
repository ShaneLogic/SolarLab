# Historical CdTe Installation Benchmarks

## Global Deployment Status
- **Total Deployed Capacity**: > 2 GW (as of reference period)

## Key Installation Examples

### Waldpolenz, Germany (First Solar)
- **Capacity**: 40 MW
- **Price**: €3,250 per kW
- **Significance**: Early large-scale CdTe deployment in Europe

### Blythe, California
- **Capacity**: 7.5 MW
- **Power Delivery Agreement**: 12 cents per kWh
- **Significance**: US utility-scale deployment reference

### Southern California Edison (First Solar Contract)
- **Capacity**: 250 MW total
- **Installation Price**: $3.50 per Watt
- **Note**: Pricing includes tax incentives
- **Significance**: Large-scale utility contract benchmark

## Variables Reference

| Variable | Type | Description |

## Domain Context
- **Technology**: CdS/CdTe thin-film solar
- **Scale**: Utility-scale projects
- **Time Period**: 2007-2008 deployment context
- **Importance**: Medium priority for historical benchmarking

## Notes
- These benchmarks represent early large-scale CdTe deployments
- Pricing reflects market conditions at time of installation
- Regional factors (incentives, regulations) significantly impact costs
- Use as relative benchmarks rather than absolute current pricing
