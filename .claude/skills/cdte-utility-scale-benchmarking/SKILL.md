# CdTe Utility-Scale Benchmarking

## When to Use
- Benchmarking CdTe utility-scale solar projects
- Evaluating project costs against historical deployments
- Establishing baseline pricing for CdS/CdTe technology
- Comparing PPA rates for utility-scale installations

## Prerequisites
- Project data records for comparison
- Understanding of CdTe/CdS technology specifications

## Benchmarking Procedure

### 1. Establish Global Deployment Context
- Compare project scale against global installed base (>2 GW deployed as of reference period)
- Verify project falls within utility-scale category

### 2. Compare Against Key Metrics
- **Pricing Benchmarks**: Compare $/W or €/kW against historical installations
- **PPA Rates**: Evaluate power purchase agreement rates (cents/kWh)
- **Project Scale**: Assess capacity relative to reference projects

### 3. Reference Historical Installations
Use the following key projects as benchmarks:
- Waldpolenz, Germany (40 MW at €3,250/kW)
- Blythe, CA (7.5 MW with 12¢/kWh PPA)
- Southern California Edison (250 MW at $3.50/W with incentives)

### 4. Output Benchmark Results
Generate comparison showing:
- Project cost per Watt/kW
- PPA rate per kWh
- Relative positioning against historical benchmarks

## Constraints
- Metrics reflect deployments up to the reference period (2007-2008 context)
- Pricing may include tax incentives or regional subsidies
- Adjust for inflation and market conditions when applying to current projects

## Output Format
Return benchmark pricing ($/W or €/kW) and PPA rates (cents/kWh) with context on how the project compares to historical installations.
