# Detailed Electrical Characterization Equations

## Variable Definitions
- j = current density (mA/cm²)
- jL = short circuit current density
- j₀ = dark saturation current
- j₀₀ = prefactor
- q = electron charge (1.602×10⁻¹⁹ C)
- V = voltage (V)
- A = diode ideality factor
- k = Boltzmann constant (8.617×10⁻⁵ eV/K)
- T = temperature (K)
- Eg = bandgap of CdTe = 1.5 eV
- Rs = series resistance (Ω·cm²)

## Temperature Dependence Analysis
- Plot Voc vs T to extrapolate to T=0K
- Ideal cell: Voc(0K) ≈ Eg
- Lower values indicate recombination losses
