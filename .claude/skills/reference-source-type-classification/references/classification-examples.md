# Detailed Classification Examples

## Journal Article Patterns

### Physical Sciences Journals
| Journal Name | Abbreviation | Example Citation |
|--------------|-------------|------------------|
| Physical Review | Phys. Rev. | Phys. Rev. **93**, 62 (1954) |
| Physical Review Letters | Phys. Rev. Lett. | Phys. Rev. Lett. **51**, 2224 (1985) |
| Journal of Applied Physics | J. Appl. Phys. | J. Appl. Phys. **53**, 5339 (1982) |
| Surface Science | Surf. Sci. | Surf. Sci. **34**, 145 (1973) |
| Solid State Communications | Solid State Commun. | Solid State Commun. **20**, 969–972 (1976) |
| Journal of Physics Society of Japan | J. Phys. Soc. Jpn. | J. Phys. Soc. Jpn. **48**, 865 (1980) |

### Key Characteristics
- Volume number is bolded
- Year in parentheses after page number
- Page number typically indicates starting page
- No publisher or location information

## Book and Book Chapter Patterns

### Book Chapters
```
Format: Author(s), in *Book Title*, ed. by Editor (Publisher, City, Year), pp. range
```

**Examples:**
- T. Tiedje, in *Hydrogenated Amorphous Silicon II*, ed. by J.D. Joannopoulos (Springer, New York, 1984), pp. 261–300
- Y. Petroff, in *Handbook on Semiconductors*, ed. by T.S. Moss, M. Balkanski, *Optical Properties of Solids*, vol. 2 (North-Holland, Amsterdam, 1980)

### Standalone Books
```
Format: Author. *Book Title* (Publisher, City, Year)
```

**Examples:**
- E.D. Palik, *Handbook of Optical Constants of Solids* (Academic Press, New York, 1985)
- B.L. Scharman, *Metal-Semiconductor Schottky Barrier Junction and Their Applications* (Plenum, New York, 1978)
- J.H. van Vleck, *The Theory of Electric and Magnetic Susceptibilities* (Oxford, 1932)

### Key Characteristics
- Book title italicized
- Publisher information required
- City/location required
- 'ed. by' indicates book chapter

## Hybrid and Edge Cases

### Edited Volumes
- May look like books but contain 'ed. by' in author field
- Treat as standalone book if entire volume is the reference

### Handbook Articles
- May appear as chapters in handbooks
- Use book chapter format

### Thesis References
- Format: Author, *Thesis* (Institution, Year)
- Example: S. Mori, *Thesis* (University of Tokyo, 1978)
- Classify as standalone reference type
