# Reference Source Type Classification

Use this skill when categorizing bibliographic entries, organizing reference databases, or determining citation format requirements.

## When to Use
- Categorizing references in a bibliography
- Determining which citation format to apply
- Building reference databases with type metadata
- Analyzing citation patterns in literature reviews

## Classification Workflow

### 1. Examine Citation Structure
Analyze the bibliographic entry for identifying patterns and keywords.

### 2. Apply Classification Rules
Use the following decision criteria:

#### Journal Articles
**Identifiers:**
- Journal abbreviation (Phys. Rev., J. Appl. Phys., Surf. Sci.)
- Volume number (often bolded)
- Single year in parentheses
- Page number or page range

**Examples:**
- *Phys. Rev. B* **31**, 805 (1985)
- *J. Appl. Phys.* **53**, 5339 (1982)
- *Surf. Sci.* **34**, 145 (1973)

#### Conference Proceedings
**Identifiers:**
- 'in' keyword followed by conference name
- Conference abbreviations (IEEE PVSC, ICALS)
- Conference year
- Single page number

**Examples:**
- in *IEEE Photovolt. Spec. Conf.* (2002), p. 600
- in *Proc. 12th ICALS*, Prague (1987)

#### Books and Book Chapters
**Identifiers:**
- 'in' keyword followed by book title (italicized)
- 'ed. by' followed by editor name(s)
- Publisher and location information
- Page range for chapters

**Examples:**
- in *Handbook on Semiconductors*, ed. by T.S. Moss (North-Holland, Amsterdam, 1980)
- in *Hydrogenated Amorphous Silicon II*, ed. by J.D. Joannopoulos (Springer, New York, 1984), pp. 261–300

#### Standalone Books
**Identifiers:**
- Book title in italics without 'in' keyword
- Publisher information
- Location and year
- No editor prefix

**Examples:**
- M. Tinkham, *Group Theory and Quantum Mechanics* (McGraw-Hill, New York, 1964)
- E.D. Palik, *Handbook of Optical Constants of Solids* (Academic Press, New York, 1985)

### 3. Verify Classification
Confirm classification by checking for conflicting indicators or hybrid types.
