# Field Strength Calculation Methods

## Domain Field Strength from Width-Bias Slope

The field strength in an anode-adjacent domain is determined from the relationship between domain width and applied bias.

### Calculation Method

1. Measure domain width (w) at various bias voltages (V)
2. Plot w vs V
3. Field strength F = dV/dw (slope of the curve)

### Comparative Values

| Domain Type | Field Strength |
|-------------|----------------|
| Cathode-adjacent | ~80 kV/cm |
| Anode-adjacent | ~135 kV/cm |

## Singular Point Analysis

### Singular Point I (Bulk)
- Represents bulk semiconductor behavior
- Solution curves normally approach this point
- Not accessible when anode-adjacent domain forms

### Singular Point II (High-Field)
- Located at high-field horizontal branch
- Remains as cathode-adjacent domain shrinks
- Key reference for domain boundary

### Singular Point III (Anode-Adjacent)
- Intersection of n1(F) and n2(F) curves
- Defines anode-adjacent domain characteristics
- Forms when field excitation competes with quenching

## Neutrality Curve Extension

When extending neutrality curves to higher fields:
- n1(F) represents field-dependent carrier concentration
- Field excitation causes n1(F) to level off or increase
- Field quenching would decrease n1(F)
- Competition between these effects signals domain transition
