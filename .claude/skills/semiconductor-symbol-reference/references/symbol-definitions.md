# Complete Symbol Definitions

## E (Uppercase)

| Symbol | Meaning |
|--------|---------|
| **E** (bold) | Electric field vector in electromagnetic radiation |
| E | Energy, elasticity (Young's) modulus |
| E₀ | Urbach parameter, unit field in electromagnetic radiation |
| EF | Fermi energy |
| EFn, EFp | Quasi-Fermi energy (electrons/holes) |
| Ec, Ev | Lower conduction and upper valence band edge |
| EqH | Quasi-hydrogen ionization energy |
| Eg | Bandgap |
| Eg,exc | Gap energy for exciton generation |
| Ed, Ea, Et | Energy of donor, acceptor, or trap |
| Ẽd, Ẽa | Distance of donor or acceptor from Ec or Ev |
| Eda | Donor-acceptor pair level energy |
| Edd | Energy of deep donors |
| EDn, EDp | Demarcation line |
| ER, Er | Relaxation energy, energy of recombination centers |
| Ea | Activation energy |
| ES, EFr | Schottky energy, Frenkel energy |
| En | Specific subband, energy of nth level |
| Eδp | Factor quantifying gap narrowing |
| Ei | Ionization energy |
| Em | Blackbody emission |
| Eμ, Eν | Energy levels for transition from ν to μ |

## e (Lowercase)

| Symbol | Meaning |
|--------|---------|
| e | Elementary charge, electron |
| e* | Static effective charge |
| **e**, **ê** (bold) | Unit electric vector of electromagnetic radiation |
| ec, es, et | Callen, Szigetti, or transverse effective charge |
| ei | Reduced index strain components (Voigt notation) |
| eik | Excitation coefficient from ith to kth level, strain components |

## F (Uppercase)

| Symbol | Meaning |
|--------|---------|
| **F** (bold) | Electric field vector |
| F | Field, Helmholtz free energy |
| F* | Average field from ionized impurities |
| F, F' | F-center, F'-center |
| Ft,l | Transverse and longitudinal field |
| Fl | Local field |
| Fi, Fe | Built-in (internal) or external field |
| Fn | Fermi integral |
| F̃ | Franz-Keldysh function |
| Fop | Screening parameter |
| F | Force |

## f (Lowercase)

| Symbol | Meaning |
|--------|---------|
| f | Distribution function (preferably Fermi-Dirac), fractional displacement |
| f₀ | Distribution function in equilibrium |
| fn, fp | Fermi function for electrons or holes |
| fi | Oscillatory strength |
| fFD, fBE | Fermi-Dirac or Bose-Einstein distribution function |
| fB | Boltzmann distribution function |

## G (Uppercase)

| Symbol | Meaning |
|--------|---------|
| G | Gibbs free energy, shear modulus, number of states |
| G̃ | Franz-Keldysh function |
| Go | Photoelectric gain factor |

## g (Lowercase)

| Symbol | Meaning |
|--------|---------|
| g | Density of states, lineshape function, Landé g-factor |
| g, gn, gp | Generation rate |
| go | Optical generation rate |

## H (Uppercase)

| Symbol | Meaning |
|--------|---------|
| **H** (bold) | Hamiltonian, magnetic field vector |
| Hs | Surface energy |
| H | Helmholtz free energy (or enthalpy), hexagonal Bravais lattice |
| H₀ | Hamiltonian for unperturbed bands |

## h (Lowercase)

| Symbol | Meaning |
|--------|---------|
| h | Planck's constant |
| ħ | h/2π (reduced Planck's constant) |
| h | Miller index |
| h• | Hole |
