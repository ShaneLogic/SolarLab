# Microcrystalline Defect Identification

## When to Use This Skill
Use this skill when analyzing semiconducting glasses or amorphous materials that exhibit:
- Evidence of precrystallization
- Formation of large number rings
- Structural deviations from true glass structure
- Potential imperfections from growth techniques

## Identify Microcrystallite Boundaries

1. **Assess Growth Conditions**
   - Determine if imperfect growth techniques were used
   - Check for formation of small crystallite nuclei

2. **Examine Boundary Formation**
   - Look for relaxed internal strain around nuclei
   - Identify micro-surfaces created as boundaries
   - Note crystal-like boundary characteristics with localized strain

3. **Classify as Boundary Defect**
   - Confirm defect originated from crystallite nucleation
   - Verify boundary-like structure exists

## Identify Small Voids

1. **Locate Large Number Rings**
   - Scan for large number rings in the glass structure
   - Examine ring centers for void presence

2. **Characterize Void Properties**
   - Measure variable size characteristics
   - Compare to crystal lattice vacancies
   - Note bond reconfiguration differences

3. **Classify as Void Defect**
   - Confirm location within large number ring centers
   - Document size variability and structural differences

## Output Classification
Report the defect classification based on:
- **Origin**: Crystallite nuclei vs. ring center formation
- **Location**: Boundary surfaces vs. ring centers
- **Characteristics**: Strain patterns, size variability, bond configuration
