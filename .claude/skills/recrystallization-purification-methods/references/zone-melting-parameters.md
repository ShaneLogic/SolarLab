# Zone Melting Parameter Guidelines

## Key Parameters

| Parameter | Typical Range | Effect |
|-----------|---------------|--------|
| Zone width | 1-10 mm | Narrower = better separation |
| Travel speed | 1-10 cm/hr | Slower = better separation |
| Number of passes | 3-20 | More passes = higher purity |
| Temperature gradient | 10-100 °C/cm | Steeper = narrower zone |

## Impurity Segregation

The segregation coefficient k = C_solid / C_melt determines behavior:
- k < 1: Impurity concentrates at end of travel (normal freezing)
- k > 1: Impurity concentrates at beginning
- k ≈ 1: Poor separation (difficult to remove)

## Practical Considerations

1. **Container selection:** Use materials that don't contaminate melt
2. **Atmosphere:** Vacuum or inert gas to prevent oxidation
3. **Starting material:** Higher purity starting material requires fewer passes
4. **End cropping:** First and last 10-20% contain concentrated impurities
