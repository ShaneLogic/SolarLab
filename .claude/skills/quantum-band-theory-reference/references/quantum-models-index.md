### Energy-Wavevector
```
E(k) = E_c + (ℏ²k²)/(2m*)
```


### Density of States
```
g(E) ∝ √(E - E_c) (3D)
g(E) ∝ constant (2D)
g(E) ∝ 1/√(E - E_c) (1D)
```
