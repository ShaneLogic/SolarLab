# Bibliography Author Range Retrieval

Use this skill when looking up specific citations by author name range in organized bibliographies or reference sections.

## When to Use
- Searching for publications by authors with surnames in a specific alphabetical range
- Locating specific bibliographic entries from structured reference lists
- Retrieving citation details for authors M-Mor (Morgenstern through Morin)
- Accessing physics research papers, thesis references, or journal articles by known author names

## Retrieval Workflow

### 1. Determine Author Name Range
Identify the alphabetical range of interest:
- Single author: Locate their surname alphabetically
- Multiple authors: Determine the range spanned
- Common ranges: A-C, D-F, G-L, M-P, Q-S, T-Z

### 2. Navigate to Bibliography Section
Access the bibliography or references section of the source document.

### 3. Locate Alphabetical Section
Find the section corresponding to the author name range.

### 4. Extract Citation Details
For each entry, retrieve the following elements:
- Author names (full list for multi-author works)
- Publication year (including disambiguators like 2000a, 2000b)
- Source (journal name, book title, or institution for theses)
- Volume number
- Page number(s)
- Additional notes (truncation, special context)

### 5. Handle Special Cases

**Multiple Works by Same Author**: Use year suffix (2000a, 2000b) to distinguish
**Thesis References**: Include institution and year
**Truncated Entries**: Note incomplete information
**Collaborative Works**: Extract all contributing authors

## Output Format

Present retrieved entries with:
- Complete author list
- Full publication details
- Clear year designation
- Source identification
