# HR ZnO Buffer Layer Deposition

## When to Use
Apply this skill when:
- Depositing TCO layer on Cu(InGa)Se2/CdS solar cells
- CdS buffer layer is thin or non-uniform
- CdS layer has pinholes that could create parallel diodes
- Need to increase open-circuit voltage (Voc)

**Do NOT use when:** The CBD-CdS layer is thick enough (performance gains not observed)

## Deposition Procedure

### 1. Deposit HR ZnO Interlayer
Before depositing the TCO layer, deposit an interlayer of undoped high-resistivity ZnO.

### 2. Set Deposition Parameters

| Parameter | Value/Range | Notes |
|-----------|-------------|-------|
| Method | RF magnetron sputtering | Standard deposition technique |
| Target | Oxide target | Pure ZnO target |
| Thickness | ~50 nm | Standard thickness |
| Resistivity | 10^5 × TCO resistivity | High resistance compared to contact |

### 3. Verify Purpose Achieved
The HR ZnO layer should:
- Mitigate effects of locally non-uniform electronic quality in Cu(InGa)Se2
- Prevent shunting from pinholes in CdS layer
- Eliminate parallel diodes formed at Cu(InGa)Se2/ZnO junctions
- Increase open-circuit voltage

## Constraints
- Performance gain is related to CdS thickness
- No benefit observed when CBD-CdS layer is thick enough
- Ensure compatibility with subsequent TCO deposition

## Expected Result
Deposited HR ZnO layer improving Voc and mitigating shunts caused by CdS non-uniformity.
