# Specialized Reference Lookup

Use this skill when searching for specific seminal works, handbooks, or technical specifications in specialized areas of physics and materials science.

## When to Use
- Researching electric and magnetic susceptibility theory
- Studying electron-hole droplets and many-body effects in semiconductors
- Analyzing surface acoustic waves and elastic wave propagation
- Retrieving crystallographic data for intermetallic phases
- Accessing product specifications for solar modules
- Finding foundational texts in specialized physics subdomains

## Lookup Workflow

### 1. Identify Research Topic
Determine the specific area of research:
- Magnetic/electric susceptibility
- Electron-hole droplets in semiconductors
- Rayleigh and Lamb waves (acoustics)
- Crystallographic data and intermetallic phases
- Solar module specifications

### 2. Search for Reference
Use the reference lookup table to find the seminal work or handbook for the topic.

### 3. Retrieve Publication Details
Obtain complete citation information including author, title, publisher, and year.

### 4. Access Source
Locate the reference through:
- Academic libraries
- Digital repositories
- Manufacturer websites (for product specs)
- Interlibrary loan services

## Reference Categories

This skill provides lookup for:

**Foundational Books**: Seminal works establishing theoretical frameworks
**Handbooks**: Comprehensive data collections for materials properties
**Product Specifications**: Technical data sheets for commercial products
**Specialized Texts**: Domain-specific reference works

## Common Search Terms

- Author name (e.g., van Vleck, Victorov, Villars)
- Book title keywords (e.g., susceptibility, droplets, waves, crystallographic)
- Domain name (e.g., acoustics, intermetallic, semiconductor)
- Product model number (e.g., ASI-TM-100)
