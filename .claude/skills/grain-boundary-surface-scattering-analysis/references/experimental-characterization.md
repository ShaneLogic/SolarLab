# Experimental Methods for Parameter Extraction

## Determining Grain Boundary Barrier Height

**Method 1: Temperature-Dependent Conductivity**
1. Measure conductivity σ(T) from low to high temperature
2. Plot ln(σ) vs. 1/T
3. Extract activation energy from slope
4. E_a = eV_b for single dominant barrier

**Method 2: Hall Effect Measurements**
1. Measure Hall mobility μ_H(T)
2. Plot ln(μ_H) vs. 1/T
3. Slope gives barrier height
4. Compare with single-crystal mobility for μ_0

**Method 3: Field-Effect Measurements**
1. Apply gate voltage to modulate carrier concentration
2. Observe threshold voltage shifts
3. Trap density at grain boundaries affects threshold

## Determining Specularity Parameter

**Method: Thickness-Dependent Mobility**
1. Prepare samples of varying thickness
2. Measure mobility for each thickness
3. Fit to μ/μ_B = f(d/λ, s)
4. Extract s from best fit

**Typical values:**
| Surface Condition | s Value |
|-------------------|----------|
| Polished, clean | 0.8 - 1.0 |
| As-deposited film | 0.3 - 0.7 |
| Rough, contaminated | 0 - 0.3 |
