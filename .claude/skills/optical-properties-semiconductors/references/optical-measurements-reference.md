# Optical Measurement Techniques

## Kramers-Kronig Analysis
Relates real and imaginary parts of dielectric function:
```
n(ω) - 1 = (2/π) P ∫[ω'κ(ω')/(ω'² - ω²)] dω'
κ(ω) = -(2ω/π) P ∫[(n(ω') - 1)/(ω'² - ω²)] dω'
```
Used to extract optical constants from reflectance data.

## Modulation Spectroscopy
Techniques that modulate an external parameter to enhance sensitivity:
- Electroreflectance (electric field modulation)
- Photoreflectance (laser modulation)
- Thermoreflectance (temperature modulation)

## Photoconductivity Measurements

### Photoconductive Gain
```
G = τ × μ × V / L²
```
Where τ is carrier lifetime, μ is mobility, V is applied voltage, L is electrode spacing.

### Responsivity
```
R = η × (eλ/hc) × G
```
Where η is quantum efficiency, λ is wavelength, h is Planck's constant, c is speed of light.
