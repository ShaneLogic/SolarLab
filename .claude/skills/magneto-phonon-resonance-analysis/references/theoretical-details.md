# Theoretical Details

## Resonance Mechanism

When the spacing between Landau levels equals the energy of longitudinal optical (LO) phonons, electrons can be transferred more efficiently by scattering with these phonons between different Landau levels. This enhanced scattering leads to more pronounced changes in magneto-resistance.

## Formula Derivation

### Resonance Condition

From the Landau level quantization:
```
En = ħωc(n + 1/2)
```

The spacing between adjacent Landau levels is:
```
ΔE = ħωc = ħ(eB/m*)
```

Resonance occurs when this equals the LO phonon energy:
```
ħωc = ħωLO
```

### Period Length

From the resonance condition:
```
ωc = ωLO
eB/m* = ωLO
B = m*ωLO/e
```

The period in 1/B space:
```
Δ(1/B) = e/(m*ωLO) × h = eh/(m*ωLO)
```

## Physical Interpretation

The periodic oscillations in magneto-resistance arise because:
1. As B varies, Landau level spacing changes
2. When spacing matches LO phonon energy, scattering is enhanced
3. Enhanced scattering changes resistivity
4. This creates oscillatory behavior with characteristic period
