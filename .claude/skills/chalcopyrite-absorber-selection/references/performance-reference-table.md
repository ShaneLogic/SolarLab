# Performance Reference Table

## Table 40.2: Highest Total Area Efficiencies

This table provides comparative performance data for chalcopyrite materials at specific band-gap values.

| Material | Band-gap (eV) | Notable Characteristics |
|----------|---------------|------------------------|
| CuInSe2 | 1.02 | Baseline chalcopyrite material |
| Cu(InGa)Se2 | 1.12 | Higher efficiency potential |
| Cu(InAl)Se2 | 1.15 | 17% efficiency achieved |
| CuInS2 | 1.53 | Optimal for single-junction |
| CuGaSe2 | 1.68 | Tandem structure applications |
| CuAlSe2 | 2.7 | Wide band-gap applications |

## Processing Notes

### CuInS2 Processing
1. Deposit with Cu-rich composition
2. CuxS second phase forms during deposition
3. Etch away excess Cu before CdS deposition
4. Required for optimal device performance

### (AgCu)(InGa)Se2 Advantages
- Band-gap tunability through Ag/Cu ratio
- Lower alloy melting temperature compared to pure Cu variants
- Enhanced Voc potential

## Reference

Data derived from experimental results; specific values may vary based on:
- Deposition method (co-evaporation, sputtering, etc.)
- Substrate temperature
- Post-deposition treatments
- Device architecture
