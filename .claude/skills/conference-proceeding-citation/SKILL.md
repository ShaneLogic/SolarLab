# Conference Proceeding Citation Format

## When to Use
Use this skill when formatting bibliographic entries for:
- Conference papers
- Proceedings publications
- Presentations at scientific meetings

**Trigger indicators:** Entry contains "in *Proc.*" or a conference name, followed by year and page information.

## Required Information
- Author(s) of the paper
- Conference name
- Year of conference
- Page numbers

## Optional Information
- Conference location (city/venue)
- Full proceedings title

## Formatting Procedure

1. **Identify the Author(s)**
   - Extract the author(s) of the conference paper
   - Example: "J. M. Woodcock"

2. **Locate the Conference Indicator**
   - Find "in" or "in *Proc.*" phrase
   - This signals the conference proceedings context

3. **Extract the Conference Name**
   - Identify the full conference name
   - Italicize the conference name
   - Examples:
     - "*25th IEEE Photovolt. Spec. Conf.*"
     - "*Proceedings 34th IEEE Photovoltaic Specialists Conference*"

4. **Check for Location**
   - Look for city or venue information
   - Example: "Philadelphia"
   - Include if available

5. **Identify the Year**
   - Extract year in parentheses
   - Example: "(1991)"

6. **Locate Page Information**
   - Find page numbers at the end
   - Use "p." for single page, "pp." for multiple pages
   - Examples: "p. 842" or "pp. 002159-002164"

7. **Construct the Citation**
   ```
   Author. in *Conference Name*, Location (Year), p./pp. Page.
   ```

## Examples

**With location:**
```
J. M. Woodcock. in *25th IEEE Photovolt. Spec. Conf.*, Philadelphia (1991), p. 842.
```

**Without location:**
```
X. Wu et al., in *Proc. 17th European Solar Energy Conf.* (2001), p. 995.
```

**With multiple pages:**
```
A. B. Smith. in *Proceedings 34th IEEE Photovoltaic Specialists Conference* (2009), pp. 002159-002164.
```

## Common Patterns
- "in" or "in *Proc.*" precedes conference name
- Conference name is italicized
- Location appears before year (if present)
- Year in parentheses
- Page information at the end
- Use "p." for single page, "pp." for range
