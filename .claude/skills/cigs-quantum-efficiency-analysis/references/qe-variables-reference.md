# QE Analysis Variables Reference

## Key Variables

### R - Total Reflection
- Definition: Fraction of incident light reflected from front surface
- Includes grid shading losses
- Measured via spectrophotometry

### α(λ) - Absorption Coefficient
- Material-specific for Cu(InGa)Se2
- Wavelength-dependent
- Increases sharply near bandgap

### W(V) - Space-Charge Width
- Voltage-dependent depletion region width
- Determined by doping and applied bias
- Increases with reverse bias

### Ldiff - Minority-Carrier Diffusion Length
- Characteristic distance carriers travel before recombining
- Material quality indicator
- Typically larger for electrons than holes

## Typical Values

- Electron diffusion length: > 1 μm in high-quality CIGS
- Space-charge width at 0V: 0.2-0.5 μm
- Absorption coefficient at 500 nm: > 10⁵ cm⁻¹
