### Complete Formula Set

#### Abrupt Junction Depletion Widths
```
lp = [2εψD/e * Nd/(Na(Na+Nd))]^0.5
ln = [2εψD/e * Na/(Nd(Na+Nd))]^0.5
```

#### Total Depletion Width
```
W = lp + ln = [2ε(ψD-V)/e * (Na+Nd)/(NaNd)]^0.5
```

#### Si-Specific Thresholds
- Critical energy for frozen-in equilibrium: > 1 eV
- Si band gap: 1.16 eV at 300 K
- Minimum minority density with frozen-in: ~10² cm⁻³

#### Linearly Graded Junction Key Relations
- Diffusion potential increases logarithmically with doping gradient
- Width varies as (Vd - V)^(1/3)
- Field has rounded (not triangular) distribution
